# React.js Project: TO-Do List

## 🔑 Key Components

### i. Hooks
- `useState` – Manage the tasks' state
- `useEffect` – Sync tasks with localStorage
- Custom hooks (if needed) for reusability

### ii. Components
- `App` – Root component
- `TodoList` – Displays list of tasks
- `TodoItem` – Represents a single task with edit/delete functionality
- `TodoForm` – Input for adding new tasks
