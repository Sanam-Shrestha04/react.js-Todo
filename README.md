# React.js Project: TO-Do List

## 🔑 Key Components

### i. Hooks
- `useState` – Manage the tasks' state
- `useEffect` – Sync tasks with localStorage

### ii. Components
- `App` – Root component
- `TodoList` – Displays list of tasks
- `TodoCard` – Represents a single task with edit/delete functionality
- `TodoInputs` – Input for adding new tasks
